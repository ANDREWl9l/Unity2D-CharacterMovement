# Unity2D-CharacterMovement
A free to use Character Movement for 2D platformer games in Unity.

![2D Pixel Image](https://imgur.com/BeMcSAK.jpg)

![2D Image](https://imgur.com/Oo52Vyw.jpg)

## Features
- Smooth movement
- Jumping
- Events for setting up animation
- 2D Physics
